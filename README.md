# Mondrian Art Project

A recreation of Piet Mondrian's famous painting using pure HTML and CSS Grid.

## About

This project replicates the iconic style of Dutch artist Piet Mondrian,
known for his abstract paintings made of colored rectangles separated
by bold black lines. Built entirely with HTML and CSS — no JavaScript,
no images, no libraries.

## Built With

- HTML5
- CSS3 (Grid Layout)
